Liquid Death

This project is presented as a campaign deck, so instead of a poster-by-poster gallery, the work lives here as an embedded PDF that can be read inside the portfolio with the same overall visual structure.

Format A PDF-based presentation that reads more like a pitch narrative than a single static execution.
Viewing Mode Embedded directly into the portfolio so the work can still be browsed as part of the same site experience.
Focus A campaign idea organized as a deck, letting concept, copy, and pacing unfold in sequence.
